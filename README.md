# ModelDiver
A vr model viewer designed in Godot

![Version: Godot Engine](https://img.shields.io/badge/Godot-v.4.6.1-blue?logo=godotengine&link=https%3A%2F%2Fgodotengine.org%2Fdownload%2Farchive%2F4.6.1-stable%2F)
[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

## Usage
### Instructions
1. Load 3D model
2. Check preview to select scale and starting position
3. Start VR
### VR Controls
Left Controller
* Joystick - Movement

Right Controller
* Joystick - Camera/Height

### Supported File Types
* GLB/glTF
